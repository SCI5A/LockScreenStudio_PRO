# LockScreenStudio PRO — Cinematic iPhone Lockscreen Creator

A professional, fully-featured **offline-first** cinematic lock screen editor that works 100% in your browser with no backend, no paid APIs, and no cloud services.

## 🎬 Features

### Core Features
- ✅ **Cinematic Preview** - Full-screen preview with professional animations
- ✅ **Background Customization** - Upload images/videos with blur, brightness, and zoom controls
- ✅ **Advanced Clock System** - Multiple styles (Large, Medium, Analog) with 12/24-hour format
- ✅ **Hijri Calendar Support** - Full Arabic Hijri date support with proper formatting
- ✅ **Multi-Notifications** - Add, edit, delete, and reorder multiple notifications
- ✅ **Audio System** - Background music + notification sounds with independent volume controls
- ✅ **Export to PNG** - High-quality screenshot export with html2canvas
- ✅ **Auto-Save** - All settings automatically saved to browser localStorage
- ✅ **Dark/Light Theme** - Full theme support with persistent storage

### Advanced Features
- 🎨 **Layout Controls** - Precise positioning for all elements
- 🎵 **Audio Preview** - Play/pause audio files before final export
- 🌙 **Hijri Date** - Automatic conversion with Arabic month names
- ⚡ **Live Clock** - Real-time clock updates every second
- 🎬 **Framer Motion** - Professional spring animations and transitions
- 📱 **Responsive Design** - Works perfectly on all devices
- 🎯 **Optical Alignment** - Smart positioning based on content

## 🚀 Getting Started

### No Installation Required!
Simply visit: **https://sci5a.github.io/LockScreenStudio_PRO/**

### How to Use
1. **Upload Background** - Add an image or video as your lock screen background
2. **Customize Clock** - Choose time format, date, and clock style
3. **Add Notifications** - Create multiple notifications with custom text
4. **Add Audio** - Upload background music and notification sounds
5. **Preview** - Click "Preview" to see the full cinematic experience
6. **Export** - Download your creation as a high-quality PNG
